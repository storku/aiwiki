## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a Scratch program that animates a sprite walking across the screen.

2. Generate a Scratch program that simulates a simple physics environment with gravity.

3. Design a Scratch program that creates a rotating 3D cube.

4. Can you create a Scratch program where two sprites chase each other around the screen?

5. Create a Scratch program that makes a sprite change its costume when the right arrow key is pressed.

6. Generate a Scratch program that allows the user to draw on the screen with their mouse.

7. Design a Scratch program that displays a sprite's x and y coordinates on the screen.

8. Create a Scratch program that plays different instruments based on where the sprite is clicked.

9. Generate a Scratch program that makes a sprite move in a circle continuously.

10. Design a Scratch program that triggers animations when two sprites collide.
