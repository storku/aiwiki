## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Verify if email@example.com has been compromised in any known data breaches or cyber attacks.

2. Control my email, email@example.com, for any data breaches. What's its current security status?

3. Have there been any incidents with my_email@example.com? Check if it's been implicated in a hack.

4. Verify the security integrity of my email_address@example.com against potential data breaches.

5. Has email@example.com been exposed to any cyber attacks or data leaks recently? Please verify.

6. Check my email, email@example.com, against any known data breaches. What are the results?

7. For email@example.com, assess if any hacked databases or data breaches have occurred.

8. What is the status of email@example.com in regard to recent data breaches or cyber attacks? Please inform.

9. Review email@example.com for any indications of compromised security in data leaks or breaches.

10. Confirm if email@example.com is secure or has been part of any data breach incidents recently.
