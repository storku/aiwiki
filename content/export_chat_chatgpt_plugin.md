## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a CSV export of my conversation with 'User333' from the past week.

2. Export the images and links shared during my chat with 'User444' in a ZIP file.

3. Generate an export of my group chat 'BestFriendsForever' in a PDF format.

4. Create an XML export of my conversation with 'User555' from yesterday.

5. Export a specific date's conversation with 'User666' in a PDF file.

6. Generate an Epub file of my conversation with 'User777' from the past month.

7. Export the shared files in my chat with 'User888' in a ZIP file.

8. Create a JSON export of my conversation with 'User999' from the first week of October.

9. Generate an RTF file of my chat thread with 'User110' from November 1st to November 30th.

10. Export my conversation with 'User220' from April 1st to May 31st in a PDF format.
