## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Start a new chess game with 3 minutes of time on each player's clock.

2. Swap the colors for the assistant and the user in the current chess game.

3. Get the SAN (Standard Algebraic Notation) representation of the move {chess_move}.

4. List the legal moves available in the current position of the chess game.

5. Start a new chess game with the assistant playing as {color} and a specific opening strategy, for example "Sicilian Defense."

6. Undo the last move made by the user and provide an alternative move suggestion.

7. Evaluate the current position on the chessboard and give a material and positional score.

8. Show a hint or a recommended move for the user in the current game position.

9. Provide a brief explanation of an opening or strategy if the user starts with an opening move like {chess_move}.

10. Make the move {chess_move} and save the current game state as a PGN file.
