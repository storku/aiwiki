## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Can you help me summarize the content of this article? [insert link]

2. I need some help analyzing this PDF: [insert link]

3. Can you tell me the main points from this PowerPoint presentation? [insert link]

4. What important information can be taken from this infographic? [insert link]

5. I'm curious about the findings in this research paper: [insert link]

6. Can you read and provide insights from this website? [insert link]

7. Please provide important points from this Google Docs file: [insert link]

8. What is the key information in this Microsoft Word document? [insert link]

9. Can you explain the content in this online resume? [insert link]

10. Help me understand this case study by summarizing the main points: [insert link]
