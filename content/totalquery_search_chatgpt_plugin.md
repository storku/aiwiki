Example prompts that utilize the plugin:

1. Search for podcasts related to {query} using these search engines: {engine1}, {engine2}.

2. Find the latest forum discussions on {query} using these search engines: {engine1}, {engine2}.

3. Locate infographics related to {query} using these search engines: {engine1}, {engine2}.

4. Discover ebooks and pdfs about {query} using these search engines: {engine1}, {engine2}.

5. Retrieve job postings related to {query} using these search engines: {engine1}, {engine2}.

6. Search for scholarly articles on {query} using these search engines: {engine1}, {engine2}.

7. Find the latest events related to {query} in {location} using these search engines: {engine1}, {engine2}.

9. Discover the latest product releases on {query} using these search engines: {engine1}, {engine2}.
