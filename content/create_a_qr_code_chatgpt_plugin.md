**Create a QR code** is a [Plugin](/wiki/plugin) for [ChatGPT](/wiki/chatgpt).              

## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Could you create a QR code for my Instagram profile, www.instagram.com/myinstagramprofile?

2. I need a QR code for this calendar event: 'Meeting with John on 25th June 2022, 3:00 pm-4:00 pm at Conference Hall'.

3. Can you generate a QR code for this Spotify playlist link: '[https://open.spotify.com/playlist/1a2b3c'](https://open.spotify.com/playlist/1a2b3c&#39;)?

4. I need a QR code for this Telegram channel: '[https://t.me/mychannel'](https://t.me/mychannel&#39;).

5. Could you create a QR code for this online document link: '[https://docs.google.com/document/d/1a2b3c'](https://docs.google.com/document/d/1a2b3c&#39;)?

6. Can you make a QR code for this Facebook event page: '[https://www.facebook.com/events/1234567890'](https://www.facebook.com/events/1234567890&#39;)?

7. I need a QR code for this LinkedIn profile: '[https://www.linkedin.com/in/username/'](https://www.linkedin.com/in/username/&#39;).

8. Please produce a QR code for this Zoom meeting link: '[https://zoom.us/j/1234567890?pwd=ab1c2d3Ef4g5H'](https://zoom.us/j/1234567890?pwd=ab1c2d3Ef4g5H&#39;).

9. Could you create a QR code for this Venmo username: '@JaneDoe'?

10. Can you generate a QR code for this app download link: '[https://play.google.com/store/apps/details?id=com.example.app'](https://play.google.com/store/apps/details?id=com.example.app&#39;)?
