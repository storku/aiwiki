## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Move the page with the ID <pageId> to a different location in my Notion workspace.

2. Find all the pages containing the keyword <keyword> in my Notion workspace.

3. Create a new database with the title <title> and properties <properties> in my Notion workspace.

4. Add a new column with the name <columnName> and type <columnType> to the database with the ID <databaseId> in my Notion workspace.

5. Duplicate the page with the ID <pageId> and title it <newTitle> in my Notion workspace.

6. List all the tasks (to-do items) in the database with the ID <databaseId> in my Notion workspace.

7. Check off a task with the ID <taskId> in the database with the ID <databaseId> in my Notion workspace.

8. Move the block with the ID <blockId> to a new position in the hierarchy of its parent page in my Notion workspace.

9. Share the page with the ID <pageId> with a user using their email <userEmail> in my Notion workspace along with <accessLevel> access level

10. Filter pages within the database with the ID <databaseId> that have tags containing <tagName> in my Notion workspace.
