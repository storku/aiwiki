**Mini Habits** is a [Plugin](/wiki/plugin) for [ChatGPT](/wiki/chatgpt).              

## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Update the description of the habit with ID 'Variable_Habit_ID' to 'Variable_New_Description'.

2. Delete the habit with ID 'Variable_Habit_ID'.

3. Update the task with ID 'Variable_Task_ID' for the habit with ID 'Variable_Habit_ID', changing its name to 'Variable_New_Task_Name', description to 'Variable_New_Description', task to 'Variable_New_Task', end date to 'Variable_New_End_Date', and frequency to 'Variable_New_Frequency'.

4. Delete the task with ID 'Variable_Task_ID' for the habit with ID 'Variable_Habit_ID'.

5. Share the details of my habit with ID 'Variable_Habit_ID' with a friend using their email 'Variable_Friends_Email'.

6. Generate a progress report for the habit with ID 'Variable_Habit_ID' from 'Variable_Start_Date' to 'Variable_End_Date'.

7. List all the habits I've successfully maintained for at least 'Variable_Days' days.

8. Create a new habit named 'Variable_Habit_Name' with the description 'Variable_Description', initial activity 'Variable_Initial_Activity', and set a reminder for this habit at 'Variable_Reminder_Time'.

9. Retrieve a summary of my habits, including total number of habits, total number of tasks completed, and overall success rate.

10. Find a suggested habit similar to the habit with ID 'Variable_Habit_ID' that I can try next.
