## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Generate a QR code for this Instagram profile: {instagram_url}.

2. Create a QR code for this Bitcoin wallet address: {bitcoin_wallet_address}.

3. I need a QR code for this discount code: {discount_code}, Valid until {expiration_date}.

4. Generate a QR code for this restaurant menu URL: {menu_url}.

5. Create a QR code for this Amazon product: {amazon_product_url}.

6. I want a QR code for sending an email to {email_address} with subject "{subject}".

7. Generate a QR code for connecting to this Bluetooth device: {device_name}, Passkey: {passkey}.

8. Create a QR code that dials this phone number: {phone_number}.

9. I need a QR code for this Zoom meeting link: {zoom_meeting_url}, Meeting ID: {meeting_id}, Passcode: {passcode}.

10. Generate a QR code for this donation link: {donation_url}.
