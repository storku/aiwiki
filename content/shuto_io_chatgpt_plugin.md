## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Send a text message to phone_number containing the content message_text.

2. Compose and send an email to email_address, using email_subject as the subject and email_body as the body.

3. Craft a new WordPress post with a title of post_title and content as post_content.

4. Carry out the command command on the server identified by hostname.

5. Forward an SMS to phone_number with the message message_text.

6. Formulate a fresh WordPress post with the title post_title and include the content post_content.

7. Initiate the command command on the server with the hostname hostname.

8. Deliver an email to email_address featuring the subject email_subject and the content email_body.

9. Initiate a new WordPress post, assigning it the title post_title and adding the content post_content.

10. Execute the specified command command on the server with the given hostname.
