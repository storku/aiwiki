**Meme Generator** is a [Plugin](/wiki/plugin) for [ChatGPT](/wiki/chatgpt).              

## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a 'Confused Math Lady' meme with the text 'Me trying to understand $Complex_Subject'.

2. Generate a 'Galaxy Brain' meme with four stages of realization related to '$Your_Idea'.

3. Make an 'Evil Kermit' meme with the text 'Me: $Responsible_Decision, Also me: $Tempting_Decision'.

4. Create a 'Batman Slapping Robin' meme with the text 'Robin: $Wrong_Statement, Batman: $Correct_Response'.

5. Generate a 'Hide the Pain Harold' meme with the text 'When $Painful_Situation happens but I have to act like everything is fine'.

6. Create an 'American Chopper Argument' meme with five panels of a heated debate about '$Controversial_Subject'.

7. Make a 'One Does Not Simply' meme with the text 'One does not simply $Difficult_Task'.

8. Generate a 'Futurama Fry' meme with the text 'Not sure if $Ambiguous_Situation or $Alternative_Situation'.

9. Create a 'Tuxedo Winnie the Pooh' meme with the text 'Me indulging in $Ordinary_Activity vs. $Fancy_Activity'.

10. Make a 'Blank Nut Button' meme with the text 'When I see $Irresistible_Option'.
