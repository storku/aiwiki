## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Undo the last move in the checkers game with ID {game_id}.

2. Can you give me a hint for my next move in the checkers game with ID {game_id}?

3. Display the list of moves taken so far in the checkers game with ID {game_id}.

4. Which pieces can be moved by the red player in the checkers game with ID {game_id}?

5. Make a move for the blue player from position {start_position} to position {end_position} in the checkers game with ID {game_id}.

6. How many kings does each player have in the checkers game with ID {game_id}?

7. Is there a forced jump for any player in the checkers game with ID {game_id}?

8. Which pieces have not yet moved in the checkers game with ID {game_id}?

9. Did Blue player win in the checkers game with ID {game_id}?

10. Suggest the top three moves for the current player in the checkers game with ID {game_id}.
