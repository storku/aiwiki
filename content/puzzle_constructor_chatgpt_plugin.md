## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Design a crossword puzzle using the words {word_list} and corresponding clues {clue_list}.

2. I need a crossword puzzle with the grid size of {grid_size} using words {word_list} and clues {clue_list}. Can you help?

3. Craft a crossword puzzle for me using these words {word_list} and their clues {clue_list}.

4. Please create a crossword puzzle with the provided words {word_list} and their associated clues {clue_list}.

5. Generate a custom crossword puzzle using the following words {word_list} and hints {clue_list}.

6. Assemble a crossword puzzle for me with a grid size of {grid_size} using words {word_list} and clues {clue_list}.

7. I'd like a crossword puzzle featuring these words {word_list} along with their clues {clue_list}. Can you create it for me?

8. Design a crossword puzzle with grid size {grid_size}, utilizing the words {word_list} and their corresponding clues {clue_list}.

9. Build a crossword puzzle for me using the words {word_list} and these clues {clue_list}.

10. Can you create a crossword puzzle with these words {word_list} and their hints {clue_list} for me?
