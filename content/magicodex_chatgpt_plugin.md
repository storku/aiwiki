## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Find the rules for the Magic: The Gathering mechanic '$MTG_Mechanic'.

2. Search for Magic: The Gathering cards with a converted mana cost of '$MTG_Converted_Mana_Cost'.

3. Find a Magic: The Gathering card with the subtype '$MTG_Subtype'.

4. Generate a random Magic: The Gathering Commander deck.

5. Get a fat pack from the Magic: The Gathering set with the name '$MTG_Set_Name'.

6. Search for Magic: The Gathering cards with a rarity of '$MTG_Rarity'.

7. Find Magic: The Gathering cards with a loyalty of '$MTG_Loyalty'.

8. Search for Magic: The Gathering cards with the set name '$MTG_Set_Name'.

9. Find Magic: The Gathering cards with the supertype '$MTG_Supertype'.

10. Search for Magic: The Gathering cards with the flavor text containing '$MTG_Flavor_Text'.
