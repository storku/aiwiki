## Actions

- **actions_sider_ai__jit_plugin**: This API has two main functionalities:
**getContent**: It allows me to retrieve the content of a specified URL. This includes web pages, PDFs, PPTs, images, Word documents, and various other document formats, as well as content from Google Drive, Google Docs, and Google Spreadsheets. It's particularly useful for accessing and providing information from specific documents or web pages.

- **apiSearch**: This function enables me to perform Google searches and fetch JSON data based on the query provided. It's useful for finding information, images, news, and other content that can be accessed via Google Search.
