## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a flashcard with the front showing an image '{image_URL}' and the back saying '{Answer}'.

2. Show me a summary of my flashcard review progress.

3. Create a flashcard related to the topic '{Topic}' with a question '{Question}' and answer '{Answer}'.

4. Add the flashcard with ID {flashcard ID} to the deck '{deck_name}'.

5. Show me a list of all the flashcards in the deck '{deck_name}'.

6. Sort my flashcards by the attribute '{attribute}' in ascending/descending order.

7. Create a flashcard using a multiple-choice format with the question '{Question}', options '{Option1}', '{Option2}', '{Option3}', and '{Option4}', and the correct answer '{Answer}'.

8. Record my practice of the flashcard with ID {flashcard ID}, and tag it with '{tag}'.

9. Search for flashcards containing the term '{term}'.

10. Generate a progress report for my flashcard reviews from the last '{time_period}'.
