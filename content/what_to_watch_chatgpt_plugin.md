**What To Watch** is a [Plugin](/wiki/plugin) for [ChatGPT](/wiki/chatgpt).              

## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Find out the premiere date of the TV show {insert show name} using what_to_watch.getPremiereDate.

2. Get a list of the top 10 shows in the genre {insert genre} using what_to_watch.getTopShowsByGenre.

3. Find out the IMDB rating of the TV show {insert show name} using what_to_watch.getIMDBRating.

4. Search for TV shows with the actor {insert actor name} using what_to_watch.searchShowsByActor.

5. Check if the show {insert show name} has been renewed or canceled using what_to_watch.getRenewalStatus.

6. Discover shows with a similar theme to {insert show name} using what_to_watch.searchShowsByTheme.

7. Get a list of popular shows from the past year using what_to_watch.getPopularShows.

8. Find out where I can stream movies related to {insert movie name} using what_to_watch.searchMovieStreamingInfo.

9. Search for TV shows suitable for kids using what_to_watch.searchKidsShows.

10. Get a list of award-winning shows to watch using what_to_watch.getAwardWinningShows.
