## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Please fetch the information from this webpage for me: {URL}

2. Can you help me extract the main text from this link: {URL}?

3. I need the key points from this article. Can you retrieve them? Here's the URL: {URL}

4. Extract and summarize the content from this webpage for me: {URL}

5. Can you access this website and fetch the main content? Here's the link: {URL}

6. I need help extracting information from this page. Here's the URL: {URL}

7. Retrieve the main text and give me a brief summary from this webpage: {URL}

8. Please access this link and extract the relevant content: {URL}

9. I'm in a hurry and can't read this article. Can you provide a summary? Here's the link: {URL}

10. Assist me in extracting the primary content from this link: {URL}
