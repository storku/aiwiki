## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Get the ticket prices for the event with the following ID: {event_id}.

2. Find the upcoming events for the venue with the following ID: {venue_id}.

3. Check the ticket availability for the event with the following ID: {event_id} on {date}.

4. Get the seating chart for the venue with the following ID: {venue_id}.

5. Retrieve a list of popular events in the following city: {city_name}.

6. Search for events happening within the following date range: {start_date} to {end_date}.

7. Find the nearest parking options for the venue with the following ID: {venue_id}.

8. Get a list of the top 10 attractions in the following city: {city_name}.

9. Check the availability of wheelchair-accessible seating for the event with the following ID: {event_id}.

10. Search for family-friendly events happening on the following date: {date}.
