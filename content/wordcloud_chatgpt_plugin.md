## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a word cloud using the most frequently mentioned words in my favorite book series, {Book_Series_Title}.

2. Generate a word cloud from the text in my most popular blog posts related to {Popular_Blog_Topic}.

3. I'd like a word cloud from the script of the movie {Movie_Title}.

4. Can you make a word cloud using the text from my recently published article on {Article_Topic}?

5. Create a word cloud using quotes from historical figures like {List_of_Historical_Figures}.

6. Generate a word cloud from the most common words in my favorite poems by {Poet_Name}.

7. Produce a word cloud using the text from my recent product description of {Product_Name}.

8. Generate a word cloud using the words from my {Month_Name} newsletter.

9. I'd like a word cloud of the most frequent words used in my eBook, {eBook_Title}.

10. Can you create a word cloud from the text in my online course titled {Course_Title}?
