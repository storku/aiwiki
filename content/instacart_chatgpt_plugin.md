## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a shopping list for a gluten-free pasta recipe.

2. Generate a shopping list for a dairy-free mac and cheese recipe, but I already have gluten-free pasta and nutritional yeast.

3. Produce a shopping list for a chicken noodle soup recipe.

4. Can you make a shopping list for a quinoa salad recipe, but I already have quinoa and chickpeas?

5. I need a shopping list for a vegetable curry recipe.

6. Create a shopping list for a shrimp taco recipe, but I already have shrimp and tortillas.

7. Generate a shopping list for a classic Caesar salad recipe.

8. Produce a shopping list for an avocado toast recipe, but I already have avocado and bread.

9. Can you make a shopping list for a falafel wrap recipe?

10. I need a shopping list for a pad thai recipe, but I already have rice noodles and peanuts.
