## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. What is the operation manual for the Crane pump model [model_number]?

2. Find the performance curve for the Crane pump with the model number [model_number].

3. Where can I find the technical datasheet for the Crane pump model [model_number]?

4. Can you provide the dimensions and drawings for the Crane pump model [model_number]?

5. I need the service manual for the Crane pump model [model_number].

6. Find the list of accessories for the Crane pump model [model_number].

7. I need the electrical information for the Crane pump model [model_number].

8. Can you provide the list of spare parts for the Crane pump model [model_number]?

9. Find the torque values and tightening sequence for the Crane pump model [model_number].

10. I need the material selection guide for the Crane pump model [model_number].
