## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Identify the top 5 trending articles in the Substack newsletter variable: newsletter name.

2. Compare the content themes of variable: Substack link 1 and variable: Substack link 2.

3. Find the latest articles discussing variable: event in Substack newsletters.

4. Discover the most influential authors in variable: niche on Substack.

5. Retrieve the highest-rated articles from the Substack newsletter variable: newsletter name in the past variable: time period.

6. Provide a content analysis and keyword summary of variable: Substack link.

7. Search for top Substack newsletters focusing on variable: industry.

8. Find articles featuring interviews with variable: person's name in Substack newsletters.

9. Discover emerging Substack newsletters in variable: category.

10. Identify the most controversial articles in the Substack newsletter variable: newsletter name.
