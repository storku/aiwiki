## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Please generate a 'terrain' style map for the coordinates [LATITUDE, LONGITUDE].

2. I'd like to see a 'hybrid' style map for the coordinates [LATITUDE, LONGITUDE].

3. Can you create a map with a zoom level of 5 for the coordinates [LATITUDE, LONGITUDE]?

4. Generate a map with a bearing of 90 degrees for the coordinates [LATITUDE, LONGITUDE].

5. Show me a map with a tilt of 60 degrees for the coordinates [LATITUDE, LONGITUDE].

6. Please generate a 'terrain' style map for LOCATION.

7. I'd like to see a 'hybrid' style map for LOCATION.

8. Can you create a map with a zoom level of 5 for LOCATION?

9. Generate a map with a bearing of 90 degrees for LOCATION.

10. Show me a map with a tilt of 60 degrees for LOCATION.
