## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. From my Google Sheets data, generate a sales report using the most suitable application in Mantium.

2. Compare multiple applications within Mantium based on my query {user_query} and present the advantages or drawbacks of each.

3. Fetch the application with the highest usage or rating from Mantium for my query {user_query}.

4. Create a visual representation of the data obtained from my query {user_query} in Mantium.

5. Retrieve Mantium applications related to the topic {specific_topic} and provide a brief description of each.

6. Filter the suitable applications in Mantium based on the programming language {programming_language} for my query {user_query}.

7. Organize all the applications in Mantium by relevance to my query {user_query} and return the top 3 results.

8. Submit my query {user_query} to multiple relevant applications within Mantium and compare the results.

9. Construct a detailed report for my project using the data gathered from Mantium applications based on my query {user_query}.

10. Download the output file or data from Mantium corresponding to the result of my query {user_query}.
