## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create an automated weekly email update with the latest weather forecast for my location.

2. Export my Trello board '[board_name]' tasks to a CSV file and save it to my Google Drive account.

3. Send a daily Slack message in the '[channel_name]' channel with the top 5 trending YouTube videos in the 'Technology' category.

4. Monitor my Twitter account for mentions, and create a daily summary of all interactions in a Google Doc.

5. Automate adding new contacts from LinkedIn to a Google Sheets file named '[spreadsheet_name]'.

6. Maintain an updated list of my expenses from my bank statement CSV file and create a monthly budget report on Google Sheets.

7. Send me an email notification when there is a new blog post published on my favorite tech blogs.

8. Create an Asana task for each new support ticket in Zendesk, and assign it to the appropriate team member.

9. Schedule and send an automated follow-up email to [email_address] after 3 days if there is no response to my initial email.

10. Automate a weekly Google Sheets summary report of my website's Google Analytics data and email it to me.
