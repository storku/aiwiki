## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. I need to find a place at {latitude, longitude}. Can you show me a map with a marker on that spot?

2. I'm planning a picnic at {latitude, longitude}. Can you create a map of that location for me?

3. I'm doing research on the area around {latitude, longitude}. Can you generate a map with that location marked?

4. I'd like to see a map highlighting {latitude, longitude} for an upcoming vacation.

5. Show me a map with a marker at {latitude, longitude}. I'm planning a geocaching adventure there.

6. I want to learn more about the surrounding area of {latitude, longitude}. Can you generate a map for me?

7. I'm writing a travel article about {latitude, longitude}. Can you create a map with that location for reference?

8. Please generate a map with a marker at {latitude, longitude}. I'm researching real estate in that area.

9. I'd like to see a map of the terrain near {latitude, longitude}. I'm planning a mountain biking trip there.

10. In an emergency, I need to find the closest location at {latitude, longitude}. Can you generate a map for me?
