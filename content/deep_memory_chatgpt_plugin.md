## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create flashcards for a summary of the following podcast episode: ${podcast_episode_link}.

2. Generate flashcards for the key concepts from this TED Talk: ${ted_talk_link}.

3. Design a deck of flashcards named '${deck_name}' from the following technical manual: ${technical_manual_link}.

4. Generate flashcards based on the keypoints of the following webinar: ${webinar_link}.

5. Create a deck of flashcards named '${deck_name}' from the following documentary transcript: ${documentary_transcript_link}.

6. Develop flashcards for this infographic: ${infographic_link}.

7. Create a deck of flashcards named '${deck_name}' from the following list of vocabulary words: ${vocabulary_list}.

8. Generate flashcards for the most important aspects of this case study: ${case_study_link}.

9. Design a deck of flashcards named '${deck_name}' for the historical events in this timeline: ${timeline_link}.

10. Create a deck of flashcards named '${deck_name}' and generate flashcards based on the following white paper: ${white_paper_link}.
