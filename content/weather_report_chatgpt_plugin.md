## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Can you provide the current weather conditions for the airport with ICAO code EHAM?

2. What is the latest METAR data for the airport with ICAO code VHHH?

3. Please provide the current weather report for the airport with ICAO code KSFO.

4. What is the METAR weather data for the airport with ICAO code SBGR?

5. Can you give me the latest weather conditions for the airport with ICAO code WSSS?

6. What is the current METAR report for the airport with ICAO code EGCC?

7. Provide the latest weather data for the airport with ICAO code KDEN.

8. What are the current weather conditions at the airport with ICAO code VIDP?

9. Can you get the METAR information for the airport with ICAO code LIRF?

10. What is the latest weather report for the airport with ICAO code SCEL?
