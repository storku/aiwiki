## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Generate ASCII art for "TEXT" using the 'caligraphy' font.

2. Can you create ASCII art for "TEXT" using the 'diamond' font?

3. I need ASCII art for "TEXT" using the 'digital' font.

4. Could you produce ASCII art for "TEXT" using the 'drpepper' font?

5. Display ASCII art for "TEXT" using the 'double' font.

6. I'd like to see ASCII art for "TEXT" using the 'lean' font.

7. Can you render ASCII art for "TEXT" using the 'modular' font?

8. I need ASCII art for "TEXT" using the 'slant' font.

9. Could you generate ASCII art for "TEXT" using the 'sub-zero' font?

10. Show me ASCII art for "TEXT" using the 'xfiles' font.
