## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a themed Spotify playlist with these songs: ${song_list}.

2. Find the best Spotify playlists for the following genres: ${genre_list}.

3. Generate a Spotify playlist inspired by these artists: ${artist_list}.

4. Create a Spotify playlist for my upcoming workout, including these songs: ${song_list}.

5. Combine these two Spotify playlists into one: ${playlist_1} and ${playlist_2}.

6. Generate a Spotify playlist for my road trip, using these artists: ${artist_list}.

7. Create a collaborative Spotify playlist with these songs: ${song_list}.

8. Make a Spotify playlist for my party, including these genres: ${genre_list}.

9. Generate a personalized Spotify playlist for my study session using these songs: ${song_list}.

10. Create a new Spotify playlist for my morning routine, featuring these artists: ${artist_list}.
