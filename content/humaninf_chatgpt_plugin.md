## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Could you restate this text in a way that sounds more human: {text}?

2. Help me rewrite this sentence in a more conversational style: {text}.

3. Make this text sound more like it was written by a person: {text}.

4. Can you transform this statement into a more human-like expression: {text}?

5. I need a rephrased version of this text that sounds less robotic: {text}.

6. Please give me a new version of this sentence that sounds more natural: {text}.

7. Reword this statement to make it sound more human and less AI-generated: {text}.

8. I'd like a more human-sounding variation of this text: {text}.

9. Change this text to sound more like something a person would say: {text}.

10. Convert this statement into a more human-like version: {text}.
