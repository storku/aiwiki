## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Begin a Tic Tac Toe game with a variable: board size board and intermediate difficulty level.

2. Make my next move in the Tic Tac Toe game at position variable: position and continue playing.

3. Proceed with the current Tic Tac Toe game and place my move in the top right corner.

4. In the ongoing Tic Tac Toe game, I'd like to play my move in the bottom left corner.

5. Start a new Tic Tac Toe game with variable: board size board and a custom difficulty level of variable: difficulty level.

6. Play my next move in the top center position in the current Tic Tac Toe game.

7. During the current Tic Tac Toe game, I want to place my move at the bottom center position.

8. Initiate a new Tic Tac Toe game with a board size of variable: board size and a random difficulty level.

9. Resume the Tic Tac Toe game by putting my move at position variable: position on the board.

10. In the ongoing Tic Tac Toe match, I would like to position my move at variable: position on the board.
