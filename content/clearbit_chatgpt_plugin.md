## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Analyze the visitor locations from Clearbit's weekly visitor report for my website.

2. Fetch revenue details for [company_domain] using Clearbit Enrichment.

3. Search for founder and executive contacts at [company_domain] using Clearbit Prospector.

4. Discover startups in the [industry] sector with a funding range of [range] using Clearbit Discovery.

5. Enrich the company information for [company_domain] using Clearbit.

6. Find the employee count of [company_domain] using Clearbit Enrichment data.

7. Search for sales and marketing contacts at [company_domain] using Clearbit Prospector.

8. Discover companies in the [industry] sector that have grown by at least [percentage] in the last year using Clearbit Discovery.

9. Enrich the person information for all email addresses within [domain_name] using Clearbit.

10. Report on the top five pages visited on my website according to Clearbit's weekly visitor report.
