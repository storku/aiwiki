## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a Spotify playlist called 'Cafe Ambiance' with these songs: {song names}.

2. Can you make a 'Romantic Dinner' playlist on Spotify including these tracks: {song names}?

3. Generate a 'Sunday Morning' playlist on Spotify with these songs: {song names}.

4. I need a 'Beach Vibes' playlist on Spotify. Please include these songs: {song names}.

5. Create an 'Epic Gaming' playlist on Spotify with these tracks: {song names}.

6. Generate a 'Throwback Hits' playlist on Spotify with these songs: {song names}.

7. Can you make a 'Movie Night' playlist on Spotify with song from these movies: {movie names}?

8. I need an 'Office Ambiance' playlist on Spotify. Please include these songs: {song names}.

9. Create a 'Pump Up the Volume' playlist on Spotify with these tracks: {song names}.

10. Generate a 'Relaxing Instrumentals' playlist on Spotify with these songs: {song names}.
