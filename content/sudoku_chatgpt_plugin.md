## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Please provide me with a new Sudoku game. I want a {difficulty} level and a {board_size} board size.

2. Create a Sudoku puzzle for me. I would like it to be {difficulty} level and have a {board_size} board size.

3. I'd like to play a Sudoku game. Please make it {difficulty} level difficulty with a {board_size} board size.

4. Generate a Sudoku puzzle for me. I prefer a {difficulty} level game with a {board_size} board size.

5. I want to try a Sudoku game. Can you create one with a {difficulty} level and a {board_size} board size?

6. For my current Sudoku game, I need to insert the number {cell_value} into cell {cell_select}.

7. In my Sudoku puzzle, I want to place the number {cell_value} in cell {cell_select}. Please update it.

8. I'd like to add the number {cell_value} to cell {cell_select} in my Sudoku game. Can you make this update?

9. Please modify my Sudoku game by placing the number {cell_value} in cell {cell_select}.

10. I need to change my Sudoku puzzle by adding the number {cell_value} in cell {cell_select}. Please update it for me.
