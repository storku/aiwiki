## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a Zap that triggers when a new comment is posted on {Your Blog Platform} and sends a notification to {Your Notification App}.

2. Generate a Zap that triggers when a new lead is added to {Your CRM} and creates a contact in {Your Contact Management App}.

3. Create a Zap that triggers when a new document is uploaded to {Your Document Management App} and adds a note in {Your Note-Taking App}.

4. Generate a Zap that triggers when a new invoice is created in {Your Accounting App} and sends a payment reminder email.

5. Create a Zap that triggers when a new appointment is scheduled in {Your Booking App} and adds it to {Your Calendar App}.

6. Generate a Zap that triggers when a new issue is created in {Your Issue Tracking App} and sends a notification to {Your Team Communication App}.

7. Create a Zap that triggers when a new event is posted on {Your Groups Platform} and creates a reminder in {Your Reminder App}.

8. Generate a Zap that triggers when a new product is added to {Your Inventory Management App} and updates the stock on {Your E-commerce Platform}.

9. Create a Zap that triggers when a new ticket is submitted in {Your Help Desk Software} and sends an acknowledgment email.

10. Generate a Zap that triggers when a new expense is logged in {Your Expense Tracking App} and adds it to {Your Accounting App}.
