**WikiDocs** is a [Plugin](/wiki/plugin) for [ChatGPT](/wiki/chatgpt).              

## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Find the top 5 highest-rated books on Wikidocs.

2. Recommend books on Wikidocs related to {topic}.

3. Update the summary of the book with the ID {book_id} on Wikidocs to {new_summary}.

4. Retrieve the revision history of the page with the ID {page_id} on Wikidocs.

5. List all the bookmarks by user {username} on Wikidocs.

6. Search for books on Wikidocs within the category {category}.

7. Create a new Wikidocs book with the title {title}, author {author}, and description {description}.

8. Search for books on Wikidocs published between {start_year} and {end_year}.

9. Show me the search results for {keyword} on Wikidocs, but only include books with a rating of {rating_threshold} or higher.

10. Add the page with the ID {page_id} to the bookmarks of user {username} on Wikidocs.
