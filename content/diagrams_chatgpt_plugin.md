## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create an entity-relationship diagram for a basic e-commerce database.

2. Draw an organizational chart to represent the hierarchy of a company's departments.

3. Can you create a Venn diagram to compare and contrast two concepts?

4. Generate a mind map to show the different aspects of a particular topic.

5. Create a state diagram to illustrate the states and transitions of a finite state machine.

6. Draw a Gantt chart to represent a project timeline with tasks and deadlines.

7. Can you create a diagram to illustrate the process of protein synthesis?

8. Generate a diagram to model the layers of the Earth's atmosphere.

9. Create a diagram to represent the stages of the software development life cycle.

10. Draw a diagram to show the structure of the solar system and the planets' orbits.
