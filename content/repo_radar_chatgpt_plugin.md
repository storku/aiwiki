## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Find me the top 5 trending repositories on GitHub in the past week.

2. Show me repositories with the highest number of issues on GitHub related to the topic {topic}.

3. Find me active repositories on GitHub with the most contributors in the field of {field}.

4. Show me the most popular repositories on GitHub created in the past month.

5. Find me the latest repositories labeled "good first issue" on GitHub related to the topic {topic}.

6. Show me repositories in {language} on GitHub that have recently been updated.

7. Find me the most watched repositories on GitHub that are related to the topic {topic}.

8. Show me the repositories with the most forks on GitHub in the category of {category}.

9. Find me the most active repositories on GitHub with the highest number of commits in the past year.

10. Show me repositories with open-source licenses on GitHub related to the topic {topic}.
