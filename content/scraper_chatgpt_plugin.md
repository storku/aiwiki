## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Extract the list of ingredients from this recipe URL.

2. Scrape all the video links from this online tutorial webpage.

3. Find all the email addresses mentioned on this contact page.

4. Extract the table of contents from this online publication.

5. Scrape the job listings from this company's career page.

6. Extract the user reviews from this product page.

7. Scrape the timestamps from this podcast episode's webpage.

8. Find all the social media links on this influencer's website.

9. Extract the FAQ section from this customer support page.

10. Scrape the available options and prices from this online booking site.
