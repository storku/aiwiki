## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a basic website with an About and Contact page, using provided HTML, CSS, and JavaScript files, and deploy it to Netlify.

2. Design a responsive single-page website with a navigation menu and deploy it to Netlify.

3. Create a simple photo gallery website using the provided images and deploy it to Netlify.

4. Develop and deploy a basic e-commerce website with a product catalog to Netlify.

5. Create and deploy a personal portfolio website with sample projects and a contact form to Netlify.

6. Design a website with a minimalistic layout including a header, footer, and content area, and deploy it to Netlify.

7. Create a landing page for a fictional app, with a download button and feature descriptions, and deploy it to Netlify.

8. Develop a simple event invitation website with RSVP functionality and deploy it to Netlify.

9. Create and deploy a restaurant website with a menu, gallery, and reservation form to Netlify.

10. Design and deploy a basic online newsletter with the provided articles and images to Netlify.
