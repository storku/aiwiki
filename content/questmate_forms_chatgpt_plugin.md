## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Create a Quest for a job search: "Design a quest for a job search process that includes steps for updating a resume, researching job openings, customizing cover letters, and preparing for interviews.

2. Set up a Quest for a gardening project: "Create a quest for planting a vegetable garden, including steps for choosing the right plants, preparing the soil, planting, watering, and harvesting.

3. Design a Quest for a self-care routine: "Create a quest for a self-care routine with daily activities focusing on mental, emotional, and physical well-being.

4. Create a Quest for a photography project: "Design a photography challenge quest with daily themes, photo editing tips, and suggestions for sharing your work.

5. Set up a Quest for a budgeting plan: "Create a quest for setting up a personal budget that includes steps for tracking expenses, setting financial goals, and finding ways to save money.

6. Design a Quest for an event planning project: "Create a quest for planning a birthday party, including steps for choosing a theme, sending invitations, organizing games and activities, and arranging food and decorations.

7. Create a Quest for a language learning course: "Design a quest for learning Spanish that includes daily lessons, vocabulary practice, and conversation exercises.

8. Set up a Quest for a health and wellness challenge: "Create a 21-day health and wellness quest with daily tasks for improving nutrition, physical activity, and mental health.

9. Design a Quest for a fundraising campaign: "Create a quest for organizing a fundraising event, including steps for selecting a cause, setting a fundraising goal, planning the event, and promoting it to potential donors.

10. Create a Quest for a decluttering project: "Design a 30-day decluttering challenge quest with daily tasks for organizing, tidying, and minimizing items in each room of your home."
