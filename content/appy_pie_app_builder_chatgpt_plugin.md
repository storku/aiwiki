## Guide

### Example Prompts

Example prompts that utilize the plugin:

1. Build an app that allows users to find the nearest ${type_of_store} based on their location.

2. Develop an app that enables users to join ${type_of_interest} interest groups and discuss topics in forums.

3. Create an app that offers customizable ${type_of_planner} planner templates and reminders.

4. Generate an app that assists users in planning their ${type_of_event} events, from venue suggestions to guest list management.

5. Design an app that provides real-time information on ${type_of_transportation} transportation schedules and delays.

6. I want an app that helps users explore ${type_of_art} art through virtual galleries and artist profiles.

7. Create an app that allows users to compare ${type_of_product} products by their features and prices.

8. Can you build an app that sends users ${type_of_challenge} challenges to complete daily?

9. Generate an app that serves as a marketplace for ${type_of_handmade_goods} handmade goods.

10. I need an app that facilitates ${type_of_quiz} quizzes and interactive learning experiences for users.
